
Full songs written in [Strudel](https://strudel.cc), a browser-based live coding music environment. Visuals powered by Hydra.

Each `.js` file is a complete track you can paste straight into the Strudel editor and hit play.

## Setup

These tracks use custom functions from [Switch Angel's Strudel scripts](https://github.com/switchangel/strudel-scripts). You'll need to install the prebake for things like `.trancegate()` and `.rlpf()` to work.

1. Download `prebake.strudel` from the repo above
2. In Strudel, open Settings (gear icon)
3. Scroll to Prebake
4. Upload the file
5. Restart Strudel

Without the prebake, leads that use `.trancegate()` will sound like continuous drones instead of gated synths. Everything else will still work.

## Usage

1. Open [strudel.cc](https://strudel.cc)
2. Paste the contents of any `.js` file into the editor
3. Ctrl+Enter to play
